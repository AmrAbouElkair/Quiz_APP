# Quiz_APP

### it's a simple Quiz_APP built using Typescript

## ## How to run

1 - Clone the repo by running (`https://github.com/AmrAbouElkair/Quiz_APP.git`)

2 - create `tsconfig.json` by using `tsc --init`

3 - in **module** set `"rootDir": "./src"` and `"outDir": "./dist"`

4 - `tsc -w` and Enjoy
